# 📦 Entrega Sprint [6] - [Guilherme]

## 🎯 O que faz:
Bot que manda mensagens automáticas ao receber um comando

## 🚀 Como rodar:
1. crie sua instancia no evolutionAPI
2. instale as dependencias do programa no terminal: 'npm install'
3. crie um arquivo .env usando o .env.example como base
4. atualize seu github com o novo arquivo
5. configure o render.com com o seu projeto no github e coloque as informações do .env
6. agora no evolutionAPI configure sua instancia com a URL do render e habilite o evnto'MESSAGES_UPSERT'
7. conecte sua instancia no whatsapp 

## 🧠 O que aprendi:
- Conceitos de automação de bot para whatsapp utilizando instancias no evolutionAPI com github e render
